# Mastra.getMCPServers()

The `.getMCPServers()` method is used to retrieve all MCP server instances that have been registered in the Mastra instance.

## Usage example

```typescript copy
mastra.getMCPServers();
```

## Parameters

This method does not accept any parameters.
