# Mastra.getTelemetry()

The `.getTelemetry()` method is used to retrieve the telemetry instance that has been configured in the Mastra instance.

## Usage example

```typescript copy
mastra.getTelemetry();
```

## Parameters

This method does not accept any parameters.
