# Mastra.getLogsByRunId()

`.getLogsByRunId()` メソッドは、特定の run ID と transport ID に対するログを取得するために使用します。このメソッドを利用するには、`getLogsByRunId` 操作をサポートするように設定されたロガーが必要です。

## 使用例

```typescript copy
mastra.getLogsByRunId({ runId: "123", transportId: "456" });
```
