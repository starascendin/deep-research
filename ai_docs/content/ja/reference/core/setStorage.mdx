# Mastra.setStorage()

`.setStorage()` メソッドは、Mastra インスタンスに使用するストレージインスタンスを設定します。このメソッドは、`MastraStorage` を1つだけ受け取ります。

## 使い方の例

```typescript copy
mastra.setStorage(
  new LibSQLStore({
    url: ":memory:"
  })
);
```

    },
  ]}

## 戻り値

このメソッドは値を返しません。
