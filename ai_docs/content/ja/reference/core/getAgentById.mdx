# Mastra.getAgentById()

`.getAgentById()` メソッドは、ID を指定してエージェントを取得します。このメソッドは、エージェントの ID を表す単一の `string` 型パラメータを受け取ります。

## 使用例

```typescript copy
mastra.getAgentById("test-agent-123");
```
