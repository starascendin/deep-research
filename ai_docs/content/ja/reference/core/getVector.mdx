# Mastra.getVector()

`.getVector()` メソッドは、名前を指定してベクターストアを取得します。このメソッドは、ベクターストア名を表す `string` 型のパラメーターを 1 つ受け取ります。

## 使い方の例

```typescript copy
mastra.getVector("testVectorStore");
```

## パラメータ

<PropertiesTable
  content={[
    {
      name: "name",

      description: "取得するベクターストアの名前。Mastra の構成に存在する有効なベクターストア名である必要があります。",
    },
  ]}
/>
