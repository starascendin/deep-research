# Mastra.getMemory()

`.getMemory()` メソッドは、Mastra インスタンスで構成されたメモリ インスタンスを取得するために使用します。

## 使用例

```typescript copy
mastra.getMemory();
```

## パラメータ

このメソッドはパラメーターを受け取りません。
