# Mastra.getWorkflow()

`.getWorkflow()` メソッドは、ID を指定してワークフローを取得します。メソッドはワークフロー ID と、任意のオプションオブジェクトを受け取ります。

## 使用例

```typescript copy
mastra.getWorkflow("testWorkflow");
```
