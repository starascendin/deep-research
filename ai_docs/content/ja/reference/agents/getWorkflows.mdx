# Agent.getWorkflows()

`.getWorkflows()`メソッドは、エージェントに設定されたワークフローを取得し、それらが関数である場合は解決します。これらのワークフローにより、エージェントは定義された実行経路を持つ複雑な多段階処理を実行することができます。



## 使い方の例

```typescript copy
await agent.getWorkflows();
```
