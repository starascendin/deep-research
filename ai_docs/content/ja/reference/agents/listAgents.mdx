# Agent.listAgents()

`.listAgents()` メソッドは、エージェントに設定されたサブエージェントを取得し、それらが関数である場合は実体化（解決）します。これらのサブエージェントによって、エージェントは他のエージェントにアクセスし、複雑な処理を実行できるようになります。

## 使用例

```typescript copy
await agent.listAgents();
```
