# Agent.getInstructions()

`.getInstructions()` メソッドは、エージェントに設定された指示を取得し、それらが関数である場合は評価して解決します。これらの指示はエージェントの挙動を導き、能力と制約を定義します。

## 使い方の例

```typescript copy
await agent.getInstructions();
```
