# Agent.getTools()

`.getTools()`メソッドは、エージェントに設定されたツールを取得し、それらが関数である場合は解決します。これらのツールはエージェントの機能を拡張し、特定のアクションの実行や外部システムへのアクセスを可能にします。



## 使用例

```typescript copy
await agent.getTools();
```
