# Run.cancel()

`.cancel()` メソッドは、ワークフロー実行をキャンセルし、処理を停止してリソースを解放します。



## 使い方の例

```typescript showLineNumbers copy
const run = await workflow.createRunAsync();

await run.cancel();
```

## 拡張された使用例

```typescript showLineNumbers copy
const run = await workflow.createRunAsync();

try {
  const result = await run.start({ inputData: { value: "initial data" } });
} catch (error) {
  await run.cancel();
}
```
