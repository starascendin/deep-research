# StepOptions

ワークフローステップの設定オプションで、変数マッピング、実行条件、その他のランタイム動作を制御します。

## 使用方法

```typescript
workflow.step(processOrder, {
  variables: {
    orderId: { step: "trigger", path: "id" },
    userId: { step: "auth", path: "user.id" },
  },
  when: {
    ref: { step: "auth", path: "status" },
    query: { $eq: "authenticated" },
  },
});
```
