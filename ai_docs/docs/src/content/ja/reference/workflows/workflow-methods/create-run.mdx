# Workflow.createRunAsync()

`.createRunAsync()` メソッドは、新しいワークフロー実行インスタンスを作成し、特定の入力データでワークフローを実行できるようにします。これは `Run` インスタンスを返す現行の API です。

<Callout>
メソッドを備えたオブジェクトを返す従来の `createRun()` メソッドについては、[Legacy Workflows](../../legacyWorkflows/createRun.mdx) セクションを参照してください。
</Callout>

## 使い方の例

```typescript copy
await workflow.createRunAsync();
```

## 拡張例

```typescript showLineNumbers copy
const workflow = mastra.getWorkflow("workflow");

const run = await workflow.createRunAsync();

    value: 10,
  },
});
```
