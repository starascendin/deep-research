# Agent.getDescription()

`.getDescription()`メソッドは、エージェントに設定された説明を取得します。このメソッドは、エージェントの目的と機能を記述した文字列を返します。



## 使用例

```typescript copy
agent.getDescription();
```



## パラメータ

このメソッドはパラメータを取りません。
