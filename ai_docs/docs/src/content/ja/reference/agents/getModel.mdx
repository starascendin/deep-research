# Agent.getModel()

`.getModel()` メソッドは、エージェントに設定された言語モデルを取得し、それが関数であれば評価して解決します。このメソッドは、エージェントの機能を支える基盤となるモデルにアクセスするために使用されます。

## 使用例

```typescript copy
await agent.getModel();
```
