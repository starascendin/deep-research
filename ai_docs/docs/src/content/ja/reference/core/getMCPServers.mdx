# Mastra.getMCPServers()

`.getMCPServers()` メソッドは、Mastra インスタンスに登録されているすべての MCP サーバーインスタンスを取得するために使用します。

## 使い方の例

```typescript copy
mastra.getMCPServers();
```

## パラメータ

このメソッドはパラメータを受け付けません。
