# Mastra.getServer()

`.getServer()` メソッドは、Mastra インスタンスで設定されたサーバー設定を取得するために使用します。

## 使い方の例

```typescript copy
mastra.getServer();
```

## パラメータ

このメソッドはパラメータを受け付けません。
