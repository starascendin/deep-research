# Mastra.getMCPServer()

`.getMCPServer()` メソッドは、論理 ID と任意のバージョンを指定して、特定の MCP サーバーインスタンスを取得します。バージョンが指定されている場合は、その論理 ID とバージョンに完全一致するサーバーを検索します。バージョンが指定されていない場合は、指定された論理 ID を持つサーバーのうち、最も新しい releaseDate のものを返します。

## 使用例

```typescript copy
mastra.getMCPServer("1.2.0");
```
