# Mastra.setTelemetry()

`.setTelemetry()` メソッドは、Mastra インスタンス内のすべてのコンポーネントに適用するテレメトリー設定を行うために使用します。このメソッドは、テレメトリー設定を表す単一のオブジェクトを受け取ります。

## 使い方の例

```typescript copy
mastra.setTelemetry({ export: { type: "console" } });
```

## 戻り値

このメソッドは値を返しません。
