# Mastra.getAgent()

`.getAgent()` メソッドは、エージェントを取得するために使用します。エージェント名を表す単一の `string` パラメータを受け取ります。

## 使用例

```typescript copy
mastra.getAgent("testAgent");
```

  content={[
