# Mastra.getWorkflows()

`.getWorkflows()` メソッドは、Mastra インスタンスで構成済みのすべてのワークフローを取得するために使用します。メソッドは、オプションの options オブジェクトを受け取ります。

## 使い方の例

```typescript copy
mastra.getWorkflows();
```
