# Mastra.getLogger()

`.getLogger()` メソッドは、Mastra インスタンスで構成されたロガーインスタンスを取得するために使用します。

## 使用例

```typescript copy
mastra.getLogger();
```

## パラメータ

このメソッドは引数を受け取りません。
