# Mastra.getStorage()

`.getStorage()` メソッドは、Mastra インスタンスで設定されたストレージインスタンスを取得するために使用します。

## 使い方の例

```typescript copy
mastra.getStorage();
```

## パラメータ

このメソッドは引数を受け取りません。
