# Mastra.getLogs()

`.getLogs()` メソッドは、特定の transport ID に対するすべてのログを取得するために使用します。このメソッドを使用するには、`getLogs` 操作をサポートするように設定された logger が必要です。

## 使用例

```typescript copy
mastra.getLogs("456");
```
